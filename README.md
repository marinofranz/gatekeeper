# Gatekeeper - Ticketing SaaS

A "plug-and-play" ticketing platform for small venues (cafes, galleries, local bands) that want to sell tickets on their own website without paying high platform fees. It features a branded checkout and a mobile-ready QR scanner for entry.

## Key Features

- Atomicity in ticket sales (no overselling)
- Custom branding for organizers
- Mobile-responsive "staff view" for scanning QR codes at the door

## Tech Stack

- Next.js
- PostgreSQL (with transactions)
- Stripe/Mollie API
- QR Logic
